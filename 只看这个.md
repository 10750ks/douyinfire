# 抖音续火花云端包

把这个压缩包解压后，上传到 GitHub 私有仓库根目录。

上传前你只需要改两个文件：

1. config.json
   - 把“好友昵称”改成你的抖音聊天列表昵称
   - 可以改 message_templates 里的续火消息

2. cookies.json
   - 用 Cookie-Editor 导出抖音 Cookie JSON
   - 把 cookies.json 里的示例内容整段替换掉

上传后进入 GitHub 仓库：

Actions -> Douyin Fire Keeper -> Run workflow

workflow 默认北京时间每天 00:10 自动执行。
