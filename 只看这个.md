# DouYinSpark-ALL 云端版

这是基于 `BTHawake/DouYinSpark-ALL` 做的云端包。

## 你只改两个文件

### 1. `cookies.json`

用 Cookie-Editor 在抖音创作者中心页面导出 JSON，然后整段替换 `cookies.json`。

### 2. `config.json`

先用昵称模式：

```json
"matchMode": "nickname"
```

然后把：

```json
"targets": ["好友原始昵称"]
```

改成你要续火的好友原始昵称。

注意：BTH 项目文档说明，昵称要在：

```text
互动管理 -> 私信管理 -> 朋友私信
```

里取，不建议用“全部”列表里的展示名。

更推荐用抖音号：

```json
"matchMode": "short_id",
"targets": ["好友抖音号"]
```

好友抖音号可以在抖音 App 好友主页查看。

## 上传 GitHub

把整个目录上传到 GitHub 私有仓库根目录，然后运行：

```text
Actions -> DouYin Spark Flow Schedule Run -> Run workflow
```

workflow 默认北京时间每天 `00:10` 自动运行。
